# Завдання — Cloud Forex Servers

## Мета завдання

Необхідно реалізувати блок тарифів Cloud Forex Servers відповідно до дизайну у Figma.
Використовуйте усі UX та optimization best practices які знаєте — це буде оцінюватись.

## Дизайн

Figma макет — _(посилання на макет)_

## Що потрібно реалізувати

- Зверстати картки тарифних планів Cloud Forex Servers згідно дизайну.
  - Кнопка покупки на кожному тарифі має містити його унікальний ід у посиланні
    (сам URL не важливий).
  - У лайв-режимі у Figma можна побачити, як виглядає дропдаун з параметрами тарифу.
- Реалізувати елемент перемикання дата-центру.
  - Тарифи повинні змінюватись при зміні ДЦ.
- Реалізувати елемент перемикання періоду покупки.
  - Динамічне оновлення цін при перемиканні періоду.

Верстка повинна повністю відповідати дизайну.

## API

Список тарифів потрібно отримувати через API:
`https://api.zomrodev.online/v1/api/proxy/`

API повертає більше тарифів, ніж потрібно для цього завдання, тому потрібно
відфільтрувати тільки **Forex** тарифи. Тарифи повинні коректно відображатись залежно
від вибраного дата-центру (доступні зараз тільки три: Польща, Нідерланди та Німеччина).

Приклад запиту на отримання тарифів:

```
func:v2.instances.order.pricelist
out:json
lang:en
page:1
page_size:999
datacenter:12,17,19
```
